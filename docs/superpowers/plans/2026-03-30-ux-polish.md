# UX Polish Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace every silent failure, raw-exception text, and abrupt state transition in the Trilho mobile app with consistent loading, error, and empty states plus smooth animated transitions.

**Architecture:** Build four shared widgets once (`AppLoading`, `AppError`, `AppEmpty`, `app_theme_constants`), then apply them to six screens. No screen is redesigned — only state-handling is improved. `AnimatedSwitcher` with 350ms fade wraps all provider `.when()` calls.

**Tech Stack:** Flutter, Riverpod (AsyncValue.when), `shimmer: ^3.0.0` (skeleton loading), native `AnimatedSwitcher` + `FadeTransition` (state transitions)

**Spec:** `docs/superpowers/specs/2026-03-30-ux-polish-design.md`

```dart
import 'package:flutter/material.dart';
import 'package:shimmer/shimmer.dart';

class AppLoading extends StatelessWidget {
  final bool _isSkeleton;
  final double? height;
  final double? width;

  const AppLoading.spinner({super.key})
      : _isSkeleton = false,
        height = null,
        width = null;

  const AppLoading.skeleton({
    super.key,
    required double this.height,
    this.width,
  }) : _isSkeleton = true;

  @override
  Widget build(BuildContext context) {
    if (!_isSkeleton) {
      return const Center(child: CircularProgressIndicator());
    }

    final cs = Theme.of(context).colorScheme;

    return Shimmer.fromColors(
      baseColor: cs.surfaceContainerHighest,
      highlightColor: cs.surface,
      child: Container(
        height: height,
        width: width ?? double.infinity,
        decoration: BoxDecoration(
          color: cs.surfaceContainerHighest,
          borderRadius: BorderRadius.circular(8),
        ),
      ),
    );
  }
}
```

```dart
import 'package:flutter/material.dart';

const Duration kAnimFast   = Duration(milliseconds: 200);
const Duration kAnimNormal = Duration(milliseconds: 350);

/// Wraps [child] in an [AnimatedSwitcher] with a fade transition.
/// Use this to animate state changes (loading → data, data → error, etc.).
Widget fadeSwitch(Widget child) {
  return AnimatedSwitcher(
    duration: kAnimNormal,
    transitionBuilder: (child, animation) =>
        FadeTransition(opacity: animation, child: child),
    child: child,
  );
}
```

```dart
import 'package:flutter/material.dart';
import 'package:flutter_riverpod/flutter_riverpod.dart';

import '../../core/models/station_model.dart';

import '../../core/widgets/app_empty.dart';

import '../map/density_marker.dart';

final _lineDetailProvider =
    FutureProvider.family<Map<String, dynamic>, String>((ref, code) async {
  final api = ref.read(apiServiceProvider);
  return api.getLineStatus(code);
});

class LineDetailScreen extends ConsumerWidget {
  final String lineCode;
  const LineDetailScreen({super.key, required this.lineCode});

  @override
  Widget build(BuildContext context, WidgetRef ref) {
    final dataAsync = ref.watch(_lineDetailProvider(lineCode));

    return Scaffold(
      appBar: AppBar(title: Text('Linha $lineCode')),
      body: fadeSwitch(dataAsync.when(
          loading: () => _buildSkeletonList(),
          error: (e, _) => AppError(
            message: 'Não foi possível carregar a linha $lineCode',
            onRetry: () => ref.invalidate(_lineDetailProvider(lineCode)),
          ),
          data: (data) {
            final status   = data['status']        as String? ?? 'Normal';
            final message  = data['statusMessage'] as String?;
            final stations = (data['stations'] as List? ?? [])
                .map((s) => StationModel.fromJson(s as Map<String, dynamic>))
                .toList();

            return Column(
              key: const ValueKey('data'),
              children: [
                // Status banner
                Container(
                  width: double.infinity,
                  color: _statusColor(status),
                  padding: const EdgeInsets.symmetric(horizontal: 16, vertical: 12),
                  child: Column(
                    crossAxisAlignment: CrossAxisAlignment.start,
                    children: [
                      Text(
                        _statusLabel(status),
                        style: const TextStyle(
                          color: Colors.white,
                          fontWeight: FontWeight.bold,
                          fontSize: 16,
                        ),
                      ),
                      if (message != null && message.isNotEmpty)
                        Padding(
                          padding: const EdgeInsets.only(top: 4),
                          child: Text(
                            message,
                            style: const TextStyle(color: Colors.white70, fontSize: 12),
                          ),
                        ),
                    ],
                  ),
                ),

                // Stations list
                Expanded(
                  child: stations.isEmpty
                      ? const AppEmpty(
                          icon: Icons.directions_transit,
                          title: 'Nenhuma estação encontrada',
                        )
                      : ListView.separated(
                          padding: const EdgeInsets.symmetric(vertical: 8),
                          itemCount: stations.length,
                          separatorBuilder: (_, __) =>
                              const Divider(height: 1, indent: 56),
                          itemBuilder: (ctx, i) {
                            final s     = stations[i];
                            final color = DensityMarker.colorForLevel(s.densityLevel);
                            return ListTile(
                              leading: Container(
                                width: 16,
                                height: 16,
                                decoration: BoxDecoration(
                                    color: color, shape: BoxShape.circle),
                              ),
                              title: Text(s.name),
                              trailing: Text(
                                DensityMarker.labelForLevel(s.densityLevel),
                                style: TextStyle(
                                    color: color, fontWeight: FontWeight.w600),
                              ),
                              onTap: () => context.push('/station/${s.id}'),
                            );
                          },
                        ),
                ),
              ],
            );
          },
        ),
      ),
    );
  }

  Widget _buildSkeletonList() {
    return Column(
      key: const ValueKey('loading'),
      children: [
        // Status banner skeleton
        const AppLoading.skeleton(height: 52),
        const SizedBox(height: 8),
        // Station tiles skeletons
        for (int i = 0; i < 5; i++) ...[
          Padding(
            padding: const EdgeInsets.symmetric(horizontal: 16, vertical: 4),
            child: AppLoading.skeleton(height: 56),
          ),
        ],
      ],
    );
  }

  Color _statusColor(String s) => switch (s) {
        'Normal'       => Colors.green.shade600,
        'ReducedSpeed' => Colors.orange,
        'Partial'      => Colors.deepOrange,
        'Suspended'    => Colors.red.shade700,
        _              => Colors.grey,
      };

  String _statusLabel(String s) => switch (s) {
        'Normal'       => 'Operação Normal',
        'ReducedSpeed' => 'Velocidade Reduzida',
        'Partial'      => 'Operação Parcial',
        'Suspended'    => 'Linha Paralisada',
        _              => s,
      };
}
```

> **Note on `fadeSwitch` + `AnimatedSwitcher`:** `AnimatedSwitcher` uses the widget's `key` to detect changes. Each branch (`loading`, `error`, `data`) needs a distinct `ValueKey` so the switcher knows when to animate. The `_buildSkeletonList()` has `key: const ValueKey('loading')` and the data `Column` has `key: const ValueKey('data')`. `AppError` and `AppEmpty` use their default keys, which differ by type from `Column`.
